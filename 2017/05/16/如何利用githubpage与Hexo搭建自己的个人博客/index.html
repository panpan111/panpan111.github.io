2017-05-16

最近想学着自己搭建一个个人博客或者个人网站的，就查阅了很多资料，资料都大同小异，但是很多教程在搭建过程中，总是会出现一些错误，以下是查阅了资料后自己搭建成功后整理的搭建过程笔记。
1、安装Node.js
登录https://nodejs.org/en/download/ 网址，找到对应的nodejs版本，下载安装，本人选择的是64位的windows installer进行下载，安装过程和其他普通软件一样，可以根据自己的情况或喜好选择安装位置，安装成功后，可在命令窗口中使用node -v检查是否安装成功。
(http://img.blog.csdn.net/20170516215406556?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMzUzMDgzMDk=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast)
2、安装git
在git官网 https://git-scm.com/download/ 中直接下载合适的git版本，安装过程同其他软件，安装后同第一步类似，可以用 git –version 检测是否安装成功，若出现相应的版本信息，则安装成功
3、注册github账号
在github官网中注册即可，注册完之后记得添加SSH Key（让github识别绑定这台机器，允许机器提交。如果是第一次使用git，在Git Bash中执行代码“ssh-keygen -t rsa -C “自己的注册邮箱地址”，执行成功后会生成两个文件id_rsa和id_rsa.pub，将后者中的内容复制到GitHub的新增的SSH key中
4、安装Hexo
在本地新建一个Blog文件夹，右击，选择Git Bash，输入npm install -g hexo,接着输入hexo init hexo初始化Hexo ,成功后会显示Start blogging with Hexo！然后使用 cd hexo
进入到hexo目录，输入指令npm install安装依赖文件以及使用hexo generate部署形成文件
